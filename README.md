# ⚗️ Science Lab Rush

A fast-paced, educational browser game designed for **secondary school students** to learn Science through action gameplay. Players must quickly sort falling science cards into the correct subject zones before time runs out.

---

## 🎮 Gameplay Overview

**Science Lab Rush** is a reaction + sorting game where players:

- Watch science-related cards fall from the top
- Click a card to select it
- Choose the correct subject zone:
  - ⚗ Chemistry  
  - ∑ Maths  
  - 🌿 Biology  
  - ⚡ Physics  
- Earn points for correct answers
- Lose lives for mistakes or missed cards
- Survive 10 increasingly fast rounds

---

## 🧠 Educational Purpose

This game helps students revise key concepts from:

- Chemistry (elements, reactions, acids & bases)
- Mathematics (formulas, geometry, algebra)
- Biology (cells, DNA, human systems)
- Physics (forces, energy, waves)

Each card also includes a **real scientific fact** to reinforce learning.

## 🎯 Game Mechanics

### Scoring
- Correct answer: +10 points (plus bonuses)
- Combo bonus: extra points for streaks
- Wrong answer: -5 points

### Lives
- Start with 3 ❤️
- Lose a life when:
  - Wrong zone selected
  - Card reaches bottom

### Rounds
- 10 total rounds
- Difficulty increases each round (faster falling speed)
